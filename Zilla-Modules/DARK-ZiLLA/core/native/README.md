# CodezillA CLI

## Overview
A powerful command-line interface (CLI) tool designed for code analysis and potentially auto-fixing. This tool provides an interactive menu system with a vintage green compute look for an engaging user experience.

## Features
- Interactive Menu System with ASCII Art and themed colors.
- Code Analysis for Current Directory.
- Code Analysis for Specific Directories and Single Files.
- AI-powered Auto-Fix (future development).
- Configuration Settings.
- Supported Languages information.

## Getting Started

### Prerequisites
- C++ Compiler (GCC/Clang)
- CMake (version 3.10 or higher)

### Building the Project

To build CodezillA, navigate to the root directory of the project and execute the following commands:

```bash
mkdir build
cd build
cmake ..
make
```
Alternatively, you can use:
```bash
cmake -B build
cmake --build build
```

### Running the Application

After a successful build, you can run the CodezillA CLI from the project root directory:

```bash
./build/codezilla
```

This will launch the interactive menu.

## Usage
The interactive menu will guide you through the available options. You can select an option by entering the corresponding number.
